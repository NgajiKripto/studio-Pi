# Zepret - Neobrutalist Digital Photobox

Zepret is an interactive online photobox application designed specifically for Gen Z. It features a bold Neobrutalist aesthetic, real-time camera processing, and AI-powered enhancements.

## 🚀 Features

- **Real-time Camera**: High-quality mirrored camera preview via WebRTC.
- **Neobrutalist UI**: High contrast, bold borders, and vibrant colors (Pink & Yellow).
- **Timed Sessions**: Choose between 5 or 10-minute packages.
- **AI Tools**: 
  - **Pose Guide**: Real-time AI suggestions for better poses.
  - **Style Assistant**: Smart frame and sticker recommendations.
  - **Social Suggester**: Automated captions and hashtags for TikTok/IG.
- **Sticker & Frame Editor**: Interactive customization of captured photos.

## 🛠 Tech Stack

- **Frontend**: Next.js 15, Tailwind CSS, Lucide React.
- **AI Engine**: Google Gemini via Firebase Genkit.
- **Design Style**: Neobrutalism.

## 🏃 Getting Started

1. Clone the repo.
2. Install dependencies: `npm install`.
3. Set up your environment variables for Genkit/Google AI.
4. Run development server: `npm run dev`.
